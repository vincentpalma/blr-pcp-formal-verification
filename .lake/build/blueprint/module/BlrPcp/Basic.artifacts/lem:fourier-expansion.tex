  written uniquely as
  \[
    g(x) = \sum_{\alpha \in \mathbb{F}_q^n} \widehat g(\alpha)\chi_\alpha(x),

  where $\widehat g(\alpha) := \langle g,\chi_\alpha\rangle$.
