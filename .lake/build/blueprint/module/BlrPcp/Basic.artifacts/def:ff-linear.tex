\begin{definition}
\label{def:ff-linear}
\lean{BlrPcp.IsLinear}
% at /Users/yzheng/Desktop/blr-pcp-formal-verification/BlrPcp/Basic.lean:22.0-30.44
\leanok
A function $f : F^n \to F$ is linear if there exists
  $a \in F^n$ such that
  \[
    f(x) = \sum_i a_i x_i
  \]
  for every $x \in F^n$.
\end{definition}
