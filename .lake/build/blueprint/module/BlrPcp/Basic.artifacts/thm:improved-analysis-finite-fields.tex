Let $F$ be a finite field. For every function
  $f : F^n \to F$, the finite-field BLR test satisfies
  \[
    \Pr[V^{f}_{\mathrm{BLR}} = 0] \ge \Delta(f,\mathrm{LIN}).
  \]
  This is the improved finite-field analysis proved via Fourier analysis.
